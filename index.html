使用Github Pages和hexo搭建博客

使用Github Pages和hexo搭建博客

相关文档地址：

  1. Hexo
  2. Github Pages

新建github仓库

登录GitHub账号，新建一个名为 username.github.io 的仓库（此处username为你的GitHub用户名）

设置github pages

github上进入刚刚新建的项目，点击settings进入设置页面，找到GitHub Pages选项，将source设置为master branch。

全局安装hexo-cli

在全局安装hexo-cli之前，请确保你已经安装了Node.js和Git^1，然后在终端执行

npm install -g hexo-cli

配置并建立工作目录

hexo安装成功之后，执行以下操作：

  1. 在目标文件夹初始化hexo

    hexo init \

  2. 进入该文件夹

    cd \

  3. 安装依赖

    npm install

文件及文件夹说明

文件 bird
_config.yml 站点配置文件
scaffolds 脚手架
source 站点源码
themes 主题文件夹
_posts

新建文章

注：若title中包含空格，需要用引号把title引起来

hexo new [layout] \

##生成静态文件

hexo generate

配置项 描述
-d, –deploy 生成后部署
-w, –watch 监听文件变化

发布草稿

hexo publish [layout] \

启动本地服务器

hexo server

配置项 描述
-p, –port 配置本地服务器端口
-s, –static Only serve static files
-l, –log Enable logger. Override logger format.

部署

hexo deploy

配置项 描述
-p, –port 配置本地服务器端口

渲染

hexo render \ [file2] …

配置项 描述
-o, –output output destination

从别的博客项目中迁移内容

hexo migrate \

清除缓存文件和生成文件（db.json、public）

hexo clean

罗列出所有的路由

hexo list \

查看hexo版本信息

hexo version
